# Hockey Team Vote Site

This is a free static website package for voting on your men's league hockey team identity.

## Files

- `index.html` — main page
- `styles.css` — page styling
- `images/` — separate SVG image files for logo and jersey placeholders

## How to use

1. Open `index.html` in your browser to preview.
2. Replace the placeholder images in `/images` with your final jersey/logo images if desired.
3. Create a Google Form for voting.
4. In `index.html`, replace the `form-placeholder` section with your Google Form iframe embed.
5. Upload the folder to GitHub Pages, Netlify, or Vercel.

## Suggested Google Form questions

1. Favorite team name
2. Favorite logo option
3. Favorite jersey design
4. Optional comments/write-ins
