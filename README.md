# Nicer Hockey Team Vote Site

This version has a more polished visual voting flow with sections for:
1. Team names
2. Logo options
3. Jersey designs
4. Google Form embed

## Where to paste the Google Form

Open `index.html` and search for:

`REPLACE ONLY THIS PLACEHOLDER BLOCK WITH YOUR GOOGLE FORM IFRAME`

Delete the full `<div class="google-form-frame">...</div>` block and paste your Google Form iframe there.
